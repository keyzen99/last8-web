広告なし

無料版でも広告ゼロ。練習中に広告で集中を壊さない。それがBeBeat.の設計思想。

サンプル精度のタイミング制御

UIが重くてもリズムは乱れない。ネイティブオーディオスレッドによる高精度エンジン。

P95 < 2.2ms

95%のステップが理想タイミングから2.2ミリ秒以内で発音

ネイティブオーディオスレッド

iOS: AVAudioEngine + hostTimeスケジューリング。Android: AudioTrack + URGENT_AUDIO優先度スレッド。UIとは独立した専用スレッドで発音タイミングを制御。

PCMアルゴリズム合成

サンプルファイルを使わず、アルゴリズムでドラム音を生成。808/909の質感を参考にしつつ、練習の邪魔にならない音色に設計。

一般アプリの10倍以上の精度

一般的なメトロノームアプリのjitterは10〜50ms程度。BeBeat.はネイティブエンジンにより2.2ms以下を実現。

よくある質問

BeBeat.とは何ですか？
BeBeat.は、起動1秒で鳴る広告なしの練習専用リズムエンジンです。8トラック×16ステップのシーケンサーで練習用ビートを即座に作成でき、ネイティブオーディオスレッドによるサンプル精度のタイミング制御を提供します。メトロノームの代わりに、より自由度の高いリズムパターンで練習できます。
メトロノームとBeBeat.の違いは？
一般的なメトロノームアプリはクリック音のみですが、BeBeat.は8トラック（Click H/L・Cymbal・HH Open/Close・Snare・Clap・Kick）×16ステップのシーケンサーを備えています。単調なクリック音ではなく、自分好みのリズムパターンを作って練習できます。また、P95 jitter < 2.2msの高精度タイミング制御を実現しています。
どの楽器の練習に使えますか？
ピアノ、ギター、ベース、ボーカル（ボイトレ）、ドラム、ダンスなど、リズムが関わるあらゆる練習に使えます。特にピアノ練習者、ギタリスト、ボーカリスト向けに設計されています。
GarageBandとの違いは？
GarageBandは音楽制作ツールであり、起動に時間がかかり機能が多すぎます。BeBeat.はGarageBandのビートモード部分だけを抜き出して極限まで軽量化した、練習専用の道具です。起動は1秒以内、すぐに練習を開始できます。
無料で使える機能は？
無料版でもリズムエンジンのほぼ全機能を使えます。8トラック×16ステップのシーケンサー、BPM 20〜200、4種類の拍子（4/4・3/4・6/8・12/8）、練習時間の自動記録が無料です。広告も一切ありません。
練習時間は自動で記録されますか？
はい。30秒以上の再生セッションは自動的に記録されます。今日の練習時間、連続練習日数、日別グラフ、BPMトレンドを確認できます。
なぜタイミングが正確なのですか？
BeBeat.はネイティブオーディオスレッド上で発音タイミングを制御しています。iOSではAVAudioEngine + hostTimeスケジューリング、AndroidではAudioTrack + 専用スレッドを使用。UIの負荷に影響されない安定したリズムを実現。P95 jitter < 2.2msは、95%のステップが理想タイミングから2.2ms以内で発音されることを示します。
オフラインで動作しますか？
はい。BeBeat.は完全にオフラインで動作します。ネットワーク通信は一切行いません。起動時もサーバーへの接続は不要です。
広告はありますか？
一切ありません。無料版でも広告は表示されません。練習中の集中を壊さないことがBeBeat.の設計思想です。
