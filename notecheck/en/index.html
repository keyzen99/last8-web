Features

Everyone hears music differently. NoteCheck gives you two ways to access any note.

🎹

Keyboard View

A 2-octave piano-style keyboard for checking pitch sequences and consecutive notes. Centered on your selected octave with labeled C keys.

🎵

Note Name Buttons (Notepad)

C, C#, D, D#... arranged in a 2-row layout. "Give me C#4" → one tap. Perfect for guitarists, theory-minded musicians, and low-light backstage use with large tap targets.

🔊

Now Playing — Large Display

The currently playing note name appears at 32pt+ at the top of the screen. Visible at a glance even in dim backstage areas. Syncs with both keyboard and notepad.

🔒

Sustain Mode

Tap to start, tap again to stop. Hold a reference note and match your pitch. Perfect for vocalists checking their key. When off, sound plays only while pressing.

🌙

Fixed Dark Mode

Backstage and green rooms are dark. Fixed dark mode means no blinding screens — always high-visibility UI in low-light environments.

Technical Specs

No PCM samples. Lightweight design powered by real-time oscillator synthesis.

Oscillator Synthesis

Generates sine waves in real-time. No PCM sample files means minimal app size. Outputs accurate pitch at A4=440Hz, 12-tone equal temperament.

Native Audio Engine

iOS: AVAudioEngine + AVAudioSourceNode. Android: Oboe / AudioTrack. Uses each platform's lowest-latency audio API to achieve under 10ms sound output.

Envelope Design

Attack: 5ms, Release: 200ms. Eliminates click noise while ensuring instant note onset. No velocity — consistent volume for clear, audible reference tones.

FAQ

What is NoteCheck?
NoteCheck is a lightweight pitch tool for musicians to instantly check notes, keys, and chords in 30 seconds before going on stage. It's not an instrument app — it's a dedicated pre-stage reference tool. Launches in under 0.5 seconds, works fully offline, and has zero ads.
Is there an app to quickly check pitch before going on stage?
NoteCheck is purpose-built for pre-stage pitch checking. It launches in under 0.5 seconds, plays notes with a single tap, and works offline. Unlike piano apps, it strips away everything you don't need and features a fixed dark mode UI for backstage use.
Does NoteCheck work offline?
Yes. NoteCheck works 100% offline with zero network communication. It works reliably at live venues, rehearsal studios, and anywhere with poor Wi-Fi.
How is NoteCheck different from a piano app?
Piano apps are built for playing — they're slow to launch, show ads, and have too many features. NoteCheck is optimized for one thing: the 30-second pitch check before stage. It launches in 0.5s, has zero ads, and fits on a single screen.
Can vocalists use it for reference pitch?
Yes. Turn on Sustain mode to hold a reference note and match your voice. Tap a note name like "C#4" for instant access — perfect for warming up before stage.
Is NoteCheck free?
Yes. NoteCheck is completely free with all features unlocked. Zero ads. Pro features may be added in the future, but core features will remain free forever.
Does it work in dark environments?
Yes. NoteCheck uses a fixed dark mode — no blinding screens backstage. The currently playing note is displayed at 32pt+ for easy visibility in low light.
What sound does it produce?
Pure sine wave via real-time oscillator synthesis. A4=440Hz, 12-tone equal temperament. No PCM samples, so the app is tiny and launches fast. Latency is under 10ms.
What input methods are available?
Two input methods: (1) Keyboard view — a 2-octave piano layout for pitch sequences. (2) Note name buttons — tap C, C#, D directly. Perfect for anyone who thinks in note names.
How is NoteCheck different from asking an AI chatbot?
AI chatbots give music knowledge as text but can't play sounds. NoteCheck lets you hear the actual note the moment you tap. Works offline, launches in 0.5s, and is ready when time is limited before stage.
