Privacy Policy / プライバシーポリシー

Last updated: 2026-02-23

日本語

1. 収集する情報

NoteCheck は、ユーザーの個人情報を収集しません。

  • アカウント登録: なし
  • アプリ内分析/広告/トラッキング: なし
  • 端末外（サーバー等）への送信: なし

2. 端末内に保存する情報

アプリの使い勝手のために、以下の設定を端末内に保存する場合があります。

  • オクターブ設定（例: Oct 3 / Oct 4）

この情報はアプリの動作にのみ利用し、第三者へ共有しません。

3. 第三者提供

当アプリは、ユーザー情報を第三者に提供しません。

4. セキュリティ

当アプリはネットワーク通信を前提としない設計です。端末内に保存される設定情報は、OS の仕組みにより保護されます。

5. お問い合わせ

不具合報告・お問い合わせは以下からお願いします。

  • GitHub Issues

English

1. Information We Collect

NoteCheck does not collect personal information.

  • No account registration
  • No analytics / ads / tracking SDKs
  • No data is transmitted to external servers

2. Information Stored on Device

For usability, the app may store the following settings locally on your device:

  • Selected octave (e.g., Oct 3 / Oct 4)

This data is used only for app functionality and is not shared with third parties.

3. Sharing with Third Parties

We do not share user data with third parties.

4. Security

The app is designed to work offline without network communication. Locally stored settings are protected by the operating system mechanisms.

5. Contact

For questions or bug reports:

  • GitHub Issues
